# Alpha knowledge base

Drop confirmed Allodium materials here as `.md` or `.txt` files.

What belongs here:
- whitepaper excerpts
- production facts
- lore notes
- faction summaries
- FAQ answers
- onboarding for new players

How Alpha uses it:
- the base persona stays compact
- matching chunks are retrieved per question
- if the docs do not contain an answer, Alpha should say that honestly

Safe content rules:
- only add confirmed information
- do not add secrets, private keys, passwords, tokens, or admin links
- prefer short factual documents over giant duplicated dumps
